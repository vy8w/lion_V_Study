문제 4. 입력 필드에서 받은 문자열을 대상으로 주어진 조건에 알맞도록 자바스크립트 프로그램을 작성하시오.

*️⃣단, onclick() 사용

*️⃣입력 필드에서 받은 값 -> ex) let input = form['name'].value;

*️⃣아이디 입력 부분이 공백이면 경고 대화상자가 뜬다.

*️⃣아이디를 5자 미만으로 작성했을 경우 경고 대화상자가 뜬다.

*️⃣패스워드 입력 부분이 공백이면 경고 대화상자가 뜬다.

*️⃣패스워드와 패스워드 검증 입력 필드에서 입력 내용이 다를 경우 경고 대화상자가 뜬다.
